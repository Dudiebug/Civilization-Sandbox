# Structured Historical Event Schema

## Minimum Version 1.0 record
A major event declares stable event ID/type/version, time, location, participants, authoritative facts, direct cause references where known, immediate state changes, player-intervention ancestry, significance, and a presentation key.

This is sufficient for factual timelines, one-line explanations, and persistent ruin links.

## Version 1.5 extension
The complete event graph may add time spans, spatial bounds, contributing conditions, typed causal edge families, evidence references, perspective records, historical-map deltas, richer significance/retention, and source-system lineage.

Events are immutable facts. Later beliefs, myths, corrections, and chronicles are separate perspective or derived records. Query depth, time, and result counts are capped.
