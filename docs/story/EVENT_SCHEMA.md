# Structured Historical Event Schema

A promotable event declares stable event ID/type/version, start/end time, spatial bounds, participants, affected systems, authoritative facts, direct causes, contributing conditions, immediate state changes, evidence references, perspective records, player-intervention ancestry, significance tier, presentation/localization keys, and source-system version.

Events are immutable facts. Later beliefs, myths, corrections, and chronicles are separate perspective/derived records. Query depth, time, and result counts are capped.
