# Chronicle Rules

- Generate from bounded event-graph queries, not final-state guessing.
- Select an explicit omniscient or believed perspective.
- Use authored localized templates and typed grammar slots.
- Validate every sentence against source facts and provenance.
- Distinguish state, cultural, and population continuity.
- Link places and ruins to the events that formed or transformed them.
- Control repetition and prose growth through significance and retention tiers.
- Live free-form LLM text is not authoritative Version 1.0 output.
