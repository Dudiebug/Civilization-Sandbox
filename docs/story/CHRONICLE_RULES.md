# Chronicle Rules

## Version 1.0 minimum
- Record major events from authoritative state rather than guessing from final state.
- Produce short factual summaries for founding, construction, shortage, migration, conflict, destruction, rebuilding, and player intervention.
- Link each summary to source event records, actors, and locations.
- Keep prose bounded and plainly distinguish player-caused events.

## Version 1.5 horizon
- Bounded event-graph queries, explicit omniscient or believed perspective, authored localized templates, typed grammar slots, provenance validation, continuity distinctions, historical maps, and richer person/settlement/civilization histories.

## Governing rule
Live free-form LLM text is never authoritative. Version 1.0 does not require the full Story Director, perspective historiography, or long-form chronicle system.
