# Primary UI Workspaces

1. World creation and rules.
2. Main HUD, time, alerts, and bookmarks.
3. Person inspector.
4. Household/home inspector.
5. Settlement/city inspector.
6. Workplace/institution/building inspector.
7. Civilization/government inspector.
8. Technology/capability portfolio.
9. Economy, inventories, prices, and logistics.
10. Diplomacy and treaty workspace.
11. War and operations workspace.
12. History, ruins, extinction, and comparison.
13. Player powers, Binding Orders, Force Outcome, and intervention log.

Each workspace must support exact source links, observer perspective, units, data age, and accessibility.
