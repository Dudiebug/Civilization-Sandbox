# Accessibility Baseline

- Full keyboard remapping and no hard-coded mouse-only domain actions.
- Scalable UI and readable minimum text.
- Pause-and-inspect for all important information.
- High-contrast and color-vision-safe presentation.
- Patterns, labels, icons, and line styles so color never carries meaning alone.
- Reduced motion, flash, shake, blood, weather opacity, and effects.
- Adjustable speech-bubble density and notification filters.
- Clear focus order and command confirmation/preview for consequential powers.
- Formal accessibility review before release.
