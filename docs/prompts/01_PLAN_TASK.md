# Plan a Playable Build

```
Act as the planner. Do not edit files.
Plan [BUILD-NN]. Manually read root and nearest scoped codex.md files, the selected build prompt, vision authority, scope coverage, relevant specifications, current game state, tests, and prior evidence.
Check docs/plans/DECISION_GATES.md. Do not invent future-build decisions.
Propose 3-8 playable vertical slices. For each: player-visible outcome, exact boundaries, tests, Unity play check, performance/persistence impact, decision dependencies, rollback, exclusions, and stop condition. Recommend a model and reasoning effort from ambiguity and consequence; do not create model configuration.
List only genuinely blocking questions, one at a time. End with: APPROVE PLAN or CHANGES.
```
