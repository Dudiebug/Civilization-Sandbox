# Plan a Task

```
Act as the planner. Do not edit files.
Plan [TASK-ID]. Read its contract, active milestone, Blueprint/ADR references, scoped codex.md files, relevant specs, current decisions, tests, and prior evidence.
Check docs/plans/DECISION_QUEUE.md. Do not invent or ask decisions that are not open for this task.
Produce small ordered milestones. For each: exact files/boundaries, behavior, tests, performance/persistence impact, decision dependencies, rollback, exclusions, and stop condition.
List only genuinely blocking questions, one at a time. End with: APPROVE PLAN or CHANGES.
```
