# Game Design Contract

## Core loop
Observe -> detect change -> inspect -> explain -> choose a stance -> watch implementation -> compare consequences.

## Player agency
- **Observe and inspect:** search, overlays, traces, history, and tracking appropriate to the active release.
- **Binding Order:** obligates the correct decision layer; prerequisites and exact implementation remain simulated.
- **Indirect intervention:** changes physical, biological, social, or informational conditions; ordinary systems propagate effects.
- **Force Outcome:** bypasses feasibility but must reconcile invariants and permanently record the intervention.

## Lean Version 1.0 baseline
Version 1.0 is a regional causal sandbox rather than the full Blueprint early-modern platform. It requires:
- persistent named people and lightweight households;
- autonomous gathering, work, hauling, construction, settlement growth, damage, and rebuilding;
- at least two societies capable of visible and explainable divergence by release;
- a bounded resource and production model sufficient for shortages, routes, trade, and conflict consequences;
- limited diplomacy, physical conflict, migration, and recovery;
- omniscient inspection, basic causal explanations, and all three player-control classes;
- reliable semantic saves and durable major-event/ruin history.

The exact world size, population target, content counts, government abstraction, technology breadth, power roster, UI layout, and long-run test duration are not fixed here. They are selected through `docs/plans/DECISION_QUEUE.md` after relevant evidence exists.

## Version 1.5 horizon
The Blueprint v2.0 baselines - including the larger worlds, 10,000-person scale gate, 18 resource classes, 16 buildings, 16 occupations, broad institutions/governments/culture/technology/diplomacy, 24 powers, and 13 UI workspaces - remain the complete early-modern horizon, not Version 1.0 release blockers.

## Content rule
Every milestone must create a connected causal slice. A disconnected collection of technical demonstrations or a large catalog of shallow systems is not progress toward release.
