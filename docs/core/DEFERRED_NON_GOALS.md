# Deferred Work and Non-Goals

## Never core
- Live LLM citizens or governments.
- One giant civilization brain.
- Manual placement of ordinary buildings, roads, lots, zones, or districts.
- Fully rendered ordinary interiors or room-by-room skyscrapers.
- Universal linear technology tree.
- Camera-dependent authoritative simulation.
- Scripted off-screen outcomes.
- Black-box ML for authoritative civilization behavior.
- Spherical core world.
- Realistic anatomical gore.

## Deferred beyond first playable
Industrialization, electricity, automobiles, aircraft, nuclear systems, computing, space, multiplayer, executable workshop mods, console certification, cloud saves, final localization breadth, final soundtrack, and final marketing telemetry.

## First-six-month prohibition
Do not build nuclear weapons, skyscrapers, modern vehicles, detailed navies, advanced constitutions, large speech libraries, final character art, interiors, multiplayer, procedural planets, or a universal technology tree.
