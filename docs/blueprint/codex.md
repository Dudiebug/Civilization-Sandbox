# Blueprint Scope

The Version 2.0 PDF is immutable project authority for the north-star game, technical principles, and the complete early-modern Version 1.5 horizon. Do not edit or replace it in ordinary work.

ADR-001 supersedes only the PDF's release numbering and immediate Version 1.0 breadth. The active milestone plan determines what is implemented now. Proposed changes beyond ADR-001 belong in another explicitly creator-approved ADR or blueprint revision.

Cite section/page references in task contracts rather than copying large passages.
