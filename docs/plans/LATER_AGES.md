# Later-Age Expansion Plan

Do not schedule these until Version 1.0 is stable. Each major family requires a new ADR, benchmark, migration plan, performance budget, and product gate.

- [ ] **1.x Foundation depth:** more early-modern crops, animals, crafts, diseases, doctrines, governments, laws, styles, speech, authoring, accessibility, and data modding.
- [ ] **2.0 Industrial transition:** steam, machine tools, factories, rail, coal systems, industrial labor, dense pollution, mass education, and larger cities.
- [ ] **3.0 Electrification:** power grids, telecommunications, precision manufacture, utilities, large services, and multicenter cities.
- [ ] **4.0 Mechanization:** automobiles, fuel networks, aircraft, mechanized armies, faster logistics, and new tactical abstractions.
- [ ] **5.0 Atomic/information:** radiation, strategic weapons, missiles, computing, electronic communication, surveillance, and deterrence.
- [ ] **6.0+ Divergent futures:** automation, biotechnology, alternate energy and social-development paths supported by later research.
- [ ] **Distant expansion:** orbital topology, celestial resources, interplanetary logistics, scale transitions, and multi-planet civilization.

Software versions expand what is possible; they never force every civilization into one global age state.
