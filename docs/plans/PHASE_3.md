# Phase 3 — World, People, Households, and Survival

**Status:** Not Started  
**Estimated duration:** 8–12 weeks

## Objective
Produce autonomous founding groups that physically survive or fail for inspectable reasons.

## Entry condition
AI Decision Laboratory gate accepted.

## Work checklist
- [ ] Generate bounded terrain, hydrology, watersheds, resources, deposits, hazards, and world validity reports.
- [ ] Create founding groups with limited knowledge and candidate-site choice.
- [ ] Create persistent person records, names, age/life state, appearance descriptors, and stable IDs.
- [ ] Create lightweight households, guardians/dependents, pooled resources, migration, and inheritance.
- [ ] Create inheritable baseline aptitudes/tendencies with controlled variation; keep politics and ideology learned.
- [ ] Create fast state, medium attitudes, slow personality, and behaviorally active traits.
- [ ] Create bounded needs, health, hunger, injury, generic disease pressure, and care access.
- [ ] Create initial occupations, work commitments, local movement, and resource reservations.
- [ ] Create starting inventories, foraging/planting/water/storage loops.
- [ ] Create temporary shelter and camp construction with real deliveries, labor, and visual stages.
- [ ] Create minimal camera-facing billboards and omniscient person/household inspectors.
- [ ] Create births, deaths, inheritance, knowledge-carrier loss, and 100-year save/replay fixtures.
- [ ] Promote structured founding, death, shortage, and intervention events into the event graph.

## Exit gate
Seed suites show groups choose sites, build shelter, obtain food/water, care for dependents, and survive or fail with complete traces; 100-year demographic save/replay passes.

## Do not build in this phase
Full cities, broad governments, diplomacy, large speech libraries, or final art.
