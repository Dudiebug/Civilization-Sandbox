**Status:** Not Started

## Entry condition

- [ ] Implement invention projects, contributors, experiments, prototypes, spillovers, adoption, diffusion, loss, rediscovery, and reverse engineering.
- [ ] Implement overlapping cultural tendencies carried by people and institutions.

- [ ] Implement learned patriotism and separate support for polity, government, law, leader, and policy.

- [ ] Implement factions, interests, organization, reform, coups, rebellion prerequisites, and post-change capacity.
- [ ] Implement civilization formation, strategic bounded goals, merger, fracture, state/culture/population extinction, and restoration claims.
- [ ] Implement Binding Orders for policy, education, research, development, migration, and strategic projects.
- [ ] Implement thin contextual speech/template system with knowledge safety and repetition control.
- [ ] Implement capability, government, culture, faction, and civilization inspectors/overlays.
- [ ] Run matched-seed divergence and non-convergence suites.
- [ ] Run peaceful reform, coercive breakdown, knowledge-loss, reverse-engineering, and restoration scenarios.

## Exit gate
Paired seed suites show causal technology, urban, cultural, and political divergence; reform and breakdown remain explainable and within performance/save budgets.

## Do not build in this phase
Industrial technology, modern constitutions, mass parties, generated scripture, or exhaustive social simulation.
