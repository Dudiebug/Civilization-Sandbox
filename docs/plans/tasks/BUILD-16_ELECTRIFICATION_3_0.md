# Prompt - Build 16: Electrification Version 3.0

Add electricity, telecommunications, precision industry, and modern utilities as physical networks that reshape settlement form, services, productivity, state capacity, and everyday life.

Player outcome: societies build distinct power and communication systems, become dependent on them, grow dense multicenter cities, and respond to outages, shortages, disasters, inequality, and political choices.

Inspect existing industrial systems and propose 3-8 playable slices. Implement one approved slice at a time. Candidate slices include generation sources, transmission/distribution, loads and prioritization, electrified workplaces and homes, telegraph/telephone communication, precision manufacture, water/sewer and modern services, grid governance, urban intensification, and cascading failures. Each slice must join physical construction, resource inputs, skilled labor, maintenance, institutions, visible networks, causal explanation, save migration, and profiling.

Do not treat electricity or communication as global bonuses. Capacity, distance, topology, reliability, ownership, and access must matter. Off-screen simulation must remain authoritative and camera-independent.

Acceptance: competing grid designs produce different development and failure patterns; a major outage causes an inspectable cascade and recovery; communication changes coordination without granting omniscience; old industrial worlds migrate; and performance remains acceptable at the approved scale.

Defer petroleum transport, aircraft, and mechanized warfare to Version 4.0 except for narrowly justified compatibility hooks.
