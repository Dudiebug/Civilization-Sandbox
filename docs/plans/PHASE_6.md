# Phase 6 — Diplomacy, Physical War, Ecosystems, and Unrestricted Powers

**Status:** Not Started  
**Estimated duration:** 14–20 weeks

## Objective
Connect civilizations through believed information, trade, treaties, conflict, ecology, disasters, ruins, and the complete first-playable player power set.

## Entry condition
Civilization divergence gate accepted.

## Work checklist
- [ ] Implement sparse contact, recognition, claims, believed information, confidence, and stale maps.
- [ ] Implement 12 diplomatic actions, proposal evaluation, communication/authority failure, and executable treaty clauses.
- [ ] Implement trade dependence, aid, migration/refugee access, alliances, coercion, espionage thin layer, war goals, and peace.
- [ ] Implement militia/guard recruitment, persistent squads, readiness, equipment, doctrine, morale, and mobilization effects.
- [ ] Implement strategic operations, route movement, physical supply, sector encounters, retreat, wounds, death, capture, and experience.
- [ ] Implement occupation, requisition, civilian response, resistance, damage, refugees, peace aftermath, and rebuilding.
- [ ] Implement environment fields, pollution, renewable/finite resources, and hazard propagation.
- [ ] Implement hybrid ecosystems: regional populations plus persistent notable/livestock/working animals.
- [ ] Implement local/functional/global extinction and recovery from remnants.
- [ ] Implement disaster lifecycle, response, recovery, and player-created hazards.
- [ ] Implement 24 first-playable powers: six inspection, six Binding Orders, six indirect interventions, six Force Outcomes.
- [ ] Implement Force Outcome preview, reconciliation, invariant checks, undo-as-command/branch, and intervention ancestry.
- [ ] Implement Tier 0–3 ruins, world-defining landmark preservation, contamination, scavenging, reclamation, and restoration.
- [ ] Implement diplomacy, war, ecosystem, extinction, ruin, and intervention inspectors/overlays.
- [ ] Run three-actor diplomacy, large-war, forced peace, forced construction, extinction/restoration, and landmark persistence scenarios.

## Exit gate
All interaction, conflict, power, ecology, and landmark scenarios pass with deterministic authority, save/replay, performance, and creator acceptance.

## Do not build in this phase
Naval/air warfare, nuclear effects, tactical player battles, advanced ecology, or modern diplomacy.
