# Creator Decision Gates

The roadmap preserves creator control without stopping for choices that can be discovered through play.

## Decision rule

Open a decision only when the next playable slice cannot proceed safely without it. Codex presents one question, concrete options, visible consequences, and a recommended default. Provisional tuning values may be used when they are easy to change and clearly labeled.

## Build-timed decisions

| Gate | Decision informed by play |
|---|---|
| Foundation | Unity/toolchain and supported development host |
| World Viewer | Initial simulation scale used for development, not final marketing scale |
| Founding Camp | Minimum person detail, starting group size, and initial terrain/resource envelope |
| Physical Settlement | Resource, occupation, building, and settlement-depth envelope |
| Divergent Societies | Number of societies and minimum cultural/institutional contrast |
| Connected Region | Trade, price, route, and migration depth |
| Conflict and Recovery | Conflict scale, violence presentation, occupation, and recovery depth |
| God Sandbox | Initial player-power set, UI organization, and explanation depth |
| Version 1.0 release | Supported scale, save horizon, hardware, platform, soak duration, content lock, title and distribution choices |
| Versions 1.1-1.5 | Which preserved early-modern depth area receives the next release |
| Later ages | Exact next age scope only after the prior release is stable |

## Never ask early

Do not ask the creator to decide the complete resource catalog, all governments, every technology, the final power roster, detailed economy, full war model, exact UI count, release date, price, or distant-age content before those decisions can be experienced in a working build.
