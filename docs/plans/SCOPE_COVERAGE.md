## Blueprint non-goals preserved

The roadmap does not make live-LLM citizens, one universal civilization brain, ordinary manual city placement, detailed ordinary interiors, a universal linear technology tree, camera-dependent authoritative simulation, scripted off-screen outcomes, black-box authoritative ML, a spherical core world, or realistic gore into core requirements.

## Coverage audit at every build gate

The build review must answer:

1. Did this build preserve every already-implemented identity property?
2. Did any deferred scope get accidentally described as cancelled?
3. Did a shortcut undermine future physical causality, autonomous execution, divergence, omniscience, layered player agency, or durable history?
4. Does every newly discovered requirement have a mapped build or preserved backlog home?
