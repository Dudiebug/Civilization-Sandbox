# START HERE

This folder is a repository-ready control system for building **Civilization Sandbox** with Codex.
It deliberately contains **no `AGENTS.md` file**. Codex is configured to use scoped `codex.md` files instead.

## Your next action

1. Put this kit at the root of a new Git repository.
2. Open `docs/plans/CURRENT_STEP.md`.
3. In Codex, select **5.6 Sol** with **Extra High** reasoning.
4. Paste `docs/prompts/01_PLAN_TASK.md` and tell it to plan `TASK-001`.
5. Do not let it implement until you approve the plan.

## Three files to use most often

- `docs/creator/QUICK_REFERENCE.md` — one-screen operating guide.
- `docs/plans/STATUS_BOARD.md` — what is done, active, blocked, or next.
- `docs/creator/MODEL_AND_PROMPT_GUIDE.md` — model selection and exact prompting method.

## Safety rule

A box is crossed off only after automated checks, independent review, evidence, and your player-visible acceptance.
The planning system itself is complete; **game-development tasks are not yet complete**.
